📚 API Документация Lenicks Bot

🔐 Аутентификация

Все запросы требуют API ключ в заголовке:

X-API-Key: ваш_api_ключ

📊 Эндпоинты

GET /api/v1/stats

Получить общую статистику бота

{
  "success": true,
  "data": {
    "servers": 1,
    "users": 50,
    "messages": 10000,
    "quotes": 25
  }
}
GET /api/v1/guilds

Получить список серверов

GET /api/v1/users/top?limit=10

Получить топ пользователей по XP

GET /api/v1/quotes?limit=10

Получить последние цитаты
