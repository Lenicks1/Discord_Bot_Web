# 🌐 Lenicks Bot Web Dashboard

Веб-панель управления для Discord бота.
Можете адаптировать сайт под себя, это просто база кода

## ✨ Возможности

- 📊 Статистика бота
- ⚙️ Настройка через сайт
- 🛡️ Модерация через веб
- 💬 Управление цитатами
- 🌙 Тёмная/Светлая тема

## 🚀 Установка

1. `pip install -r requirements.txt`
2. Настройте `.env`
3. `python app.py`

## 🌐 Доступ

Откройте `http://localhost:5000`
